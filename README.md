# Personal Homepage Project

This project is a personal homepage that includes a digital avatar chat area. It is structured to provide a clean and interactive user experience.

## Project Structure

```
personal-homepage
├── src
│   ├── index.html          # Main HTML file for the personal homepage
│   ├── css
│   │   ├── style.css       # Main styles for the homepage
│   │   └── chat.css        # Styles specific to the chat area
│   ├── js
│   │   ├── main.js         # Main JavaScript for homepage interactions
│   │   └── chat.js         # JavaScript for chat functionality
│   └── assets
│       └── fonts           # Custom fonts for the homepage
├── package.json            # npm configuration file
└── README.md               # Project documentation
```

## Features

- **Responsive Design**: The homepage is designed to be responsive and user-friendly across various devices.
- **Chat Functionality**: Users can interact with a digital avatar through a chat interface.
- **Custom Fonts**: The project includes custom fonts to enhance the visual appeal.

## Usage

1. Clone the repository to your local machine.
2. Navigate to the project directory.
3. Open `src/index.html` in your web browser to view the homepage.
4. Use the chat area to interact with the digital avatar.

## Installation

To install the necessary dependencies, run:

```
npm install
```
